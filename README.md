# Heterodyna Seo Wordpress Plugin

## English

**Short repository description:**

Open-source WordPress SEO toolkit with AI automation, technical SEO, sitemap, robots.txt, structured data, PageSpeed monitoring, redirects, 404 tracking, content intelligence, SERP analysis and multilingual admin interface.

**README description:**

Heterodyna SEO is an open-source SEO toolkit for WordPress.

The plugin provides a central SEO management panel for metadata, indexation rules, technical SEO files, sitemap generation, social metadata, structured data, PageSpeed Insights monitoring, redirects, 404 tracking, content analysis, SERP intelligence, AI-assisted SEO workflows and social media publishing tools.

This open-source build has commercial license activation and remote status reporting disabled. Plugin modules are available without a license key. External requests may still be used only when the user configures or runs specific features such as PageSpeed Insights, Site Kit, social media integrations, sitemap fetching or URL imports.

The project is released under the **GPL-2.0-or-later** license.
When using, modifying or redistributing the code, the original license and author attribution must be preserved.

Author: **Heterodyna Team Software**.
